# Biomimetic Vibrissal Sensing for Smart Sewer Robots

Tactile sensing systems inspired by rodent whisker (vibrissal) arrays for autonomous navigation in unstructured environments.

## Overview

This project develops biomimetic vibrissal sensors for robotic systems operating in GPS-denied, visually degraded environments such as sewer infrastructure. The sensing approach draws on biological whisker systems — specifically the active tactile sensing strategies of rodents — to enable robust perception where conventional sensors (cameras, LiDAR) are unreliable.

This work contributes to both **robotics engineering** (novel sensing modalities for infrastructure inspection) and **philosophy of cognitive science** (the role of embodiment and active perception in intelligent behavior).
